## Если нужно прямо по одной задаче

Например:

- нужна задача 1 по алгоритмам — бери функцию `next_greater_height`
- нужна задача 4 по SQL — бери 4-й запрос из `sql_solutions.sql`
- нужна задача 8 по ML — бери `task8()` из `ml_tasks.py`
